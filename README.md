# Nginx-config-creator
Program help to create basic nginx configs \
This cli program can help to create basic nginx configs

## Start
#### Easy Start
1. `wget https://github.com/Sesdear/Nginx-config-creator/releases/download/1.5.0/NginxConfigCreator`
2. `chmod +x ./NginxConfigCreator && ./NginxConfigCreator`
#### Build
1. `git clone https://github.com/Sesdear/Nginx-config-creator.git && cd Nginx-config-creator`
2. `mkdir build && cd build && cmake .. && make -j$(nproc)`
3. `chmod +x NginxConfigCreator`
4. `./NginxConfigCreator`
